# BMI 계산기

키와 몸무게를 입력받아 BMI(체질량지수)를 계산하고, 체중 상태를 분류해주는 간단한 파이썬 GUI 프로그램입니다.

## 실행 방법

```
py bmi_calculator.py
```

Python이 정식으로 설치되어 있다면 아래 명령도 사용할 수 있습니다.

```
python bmi_calculator.py
```

별도 라이브러리 설치 없이 Python 표준 라이브러리인 `tkinter`만 사용합니다.

## 사용 방법

1. "키 (cm)"와 "몸무게 (kg)" 입력창에 값을 입력합니다.
2. "계산하기" 버튼을 클릭합니다.
3. 화면에 BMI 값과 분류 결과가 표시됩니다.

## BMI 계산 공식

```
BMI = 몸무게(kg) / (키(m) x 키(m))
```

## 분류 기준 (대한비만학회 기준)

| BMI 범위      | 분류    |
|--------------|---------|
| 18.5 미만     | 저체중   |
| 18.5 ~ 22.9   | 정상    |
| 23.0 ~ 24.9   | 과체중   |
| 25.0 ~ 29.9   | 비만    |
| 30.0 이상     | 고도비만 |

## 예시

- 키 170cm, 몸무게 65kg → BMI 22.5 (정상)
